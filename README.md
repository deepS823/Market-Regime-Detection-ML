# Market Regime Detection using Machine Learning

This project uses ML to classify stock market conditions into Risk-On and Risk-Off using NIFTY and India VIX.

## Highlights
- Achieved ~94% accuracy
- Used RSI, Moving Averages, VIX
- Built a risk-based trading strategy
- Real-time market prediction

## Output
- Risk-On / Risk-Off signals
- Strategy vs Market performance

## Tools
Python, yfinance, ta, sklearn, xgboost
